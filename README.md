# angular-pipe-safe-dom

### Usage

```html
<div [innerHtml]="content | safeDom">
  <!--   content -->
</div>

<object [data]="content | safeDom">
  <!--   content -->
</object>

<embed [src]="content | safeDom">
  <!--   content -->
</embed>
```
